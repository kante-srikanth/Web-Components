# Web-components
* Fetch random users when input is provided.
* Implemented using web components and randomuser.me API.

### Tech stack:
* Web components
* Javascript
* CSS
* HTML

### STEPS:
1. Clone the repo
2. Launch index.html
3. Provide input and press enter
